# スマホで確認する方法

## 方法1: ローカルネットワーク経由（推奨）

### 1. 開発サーバーを起動（ローカルネットワーク公開）

```bash
npm run dev:mobile
```

または

```bash
next dev -H 0.0.0.0
```

### 2. PCのIPアドレスを確認

**Windows:**
```powershell
ipconfig
```
「IPv4 アドレス」を確認（例: `192.168.1.100`）

**macOS/Linux:**
```bash
ifconfig
```
または
```bash
ip addr show
```

### 3. スマホからアクセス

1. PCとスマホが同じWi-Fiネットワークに接続されていることを確認
2. スマホのブラウザで以下のURLにアクセス：
   ```
   http://[PCのIPアドレス]:3000
   ```
   例: `http://192.168.1.100:3000`

### 4. PWAとしてインストール（オプション）

スマホのブラウザで：
- **iOS Safari**: 共有ボタン → 「ホーム画面に追加」
- **Android Chrome**: メニュー → 「アプリをインストール」

## 方法2: ngrokを使用（外部からアクセス可能）

### 1. ngrokをインストール

```bash
npm install -g ngrok
```

### 2. 開発サーバーを起動

```bash
npm run dev
```

### 3. 別のターミナルでngrokを起動

```bash
ngrok http 3000
```

### 4. ngrokのURLをスマホで開く

ngrokが表示するURL（例: `https://xxxx.ngrok.io`）をスマホのブラウザで開く

## 方法3: Expo Go（React Nativeが必要）

現在のプロジェクトはNext.js（Web）ベースのため、Expo Goで直接動かすことはできません。

Expo Goを使う場合は、以下のいずれかの方法が必要です：

1. **Expo Routerに移行**（大きな変更が必要）
2. **React Native版を作成**（別プロジェクト）

## トラブルシューティング

### スマホからアクセスできない場合

1. **ファイアウォールの確認**
   - Windowsファイアウォールでポート3000を許可
   - または一時的にファイアウォールを無効化してテスト

2. **ネットワークの確認**
   - PCとスマホが同じWi-Fiに接続されているか確認
   - モバイルデータではなくWi-Fiを使用

3. **IPアドレスの確認**
   - PCのIPアドレスが正しいか確認
   - `ipconfig`（Windows）または`ifconfig`（macOS/Linux）で再確認

4. **ポートの確認**
   - ポート3000が使用可能か確認
   - 他のアプリがポート3000を使用していないか確認

### HTTPSが必要な場合

PWAの一部機能（カメラ、位置情報など）はHTTPSが必要です。その場合は：

1. **ngrokを使用**（自動でHTTPS）
2. **mkcertでローカル証明書を作成**

## 推奨方法

**開発中**: 方法1（ローカルネットワーク経由）が最も簡単です。
**デモ・共有**: 方法2（ngrok）が便利です。
